Block and Inline Block Element

Block element always startson a new line and takes up the full width of a page, from left to right. A block-level element can take up one line or multiple lines and has a line break before and after the element. And inline blick element does not cause a line break and does not take up the full width of a page, only the space bounded by its opening and closing tag.
